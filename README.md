# 📊 Social Media Reach Analysis using Python

A complete Data Analytics and Machine Learning project that analyzes social media performance using Python. This project explores audience engagement, identifies factors affecting post reach, and predicts the reach of future social media posts using a Linear Regression model.

---

## 📌 Project Overview

Social media platforms generate a vast amount of engagement data every day. Understanding this data helps businesses and content creators optimize their content strategy and maximize audience reach.

This project performs:

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Data Visualization
* Engagement Rate Analysis
* Platform and Content Performance Analysis
* Reach Prediction using Machine Learning

---

## 🚀 Features

* 📂 Load social media dataset from a CSV file
* 🧹 Remove missing values and duplicate records
* 📊 Generate descriptive statistics
* 🔥 Correlation Heatmap
* 📈 Reach Distribution Analysis
* 👍 Likes Distribution Analysis
* 📱 Platform-wise Reach Comparison
* 📝 Content Type Performance Analysis
* 📦 Box Plot Analysis
* 🎯 Scatter Plot Analysis
* 🔗 Pair Plot Visualization
* 💡 Engagement Rate Calculation
* 🏆 Top 10 Highest-Reach Posts
* ⏰ Best Posting Time Analysis
* 📅 Day-wise Reach Analysis
* 🤖 Linear Regression Model for Reach Prediction
* 📉 Model Evaluation using MAE, MSE, RMSE, and R² Score
* 💾 Export Processed Dataset

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Social_Media_Reach_Analysis.git
```

Move into the project folder:

```bash
cd Social_Media_Reach_Analysis
```

Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

---

## ▶️ How to Run

Place the dataset (`Social_Media_Reach_Analysis_NovaTech.csv`) in the project folder and run:

```bash
python social_media_reach_analysis.py
```
